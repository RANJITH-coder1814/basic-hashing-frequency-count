# 🔢 Basic Hashing (Frequency Count) in C++

This project demonstrates **Basic Hashing** using C++.
Hashing is used to efficiently count the frequency of elements in an array.

## 🚀 Features

* Simple hashing implementation
* Counts frequency of numbers
* Beginner-friendly C++ program

## 🧠 How It Works

1. Input an array of numbers.
2. Create a hash array to store frequencies.
3. Increment the index corresponding to each number.
4. Print the frequency of each element.

## 💻 Example

Input:

```
5
1 2 2 3 1
```

Output:

```
1 -> 2
2 -> 2
3 -> 1
```

* C++
